\section{\MakeUppercase{FEASIBILITY ANALYSIS}}
\subsection{Technical Feasibility}
The project uses Python, Pytorch library, Conformer-Transformer architecture which requires adequate expierence in transformer in audio and LLMs. 

\subsection{Operational Feasibility}
The system is a speech-to-speech system which will be interfaced neatly with a simple web based GUI for convenience.

\subsection{Economic Feasibility}
This project requires the need of online computing services for training the ASR transformer model by scratch. Accounting for online computing services for few months on google colab, kaggle or other competitive places, the project is economically feasible. Due to lack of hardware and other requirements, the project is relatively low cost.

\subsection{Time Feasibility}
The project is achievable within academic timeline. The milestones can be divided into ASR model and LLM in two equal halves and can be achieved with proper evaluation and testing.
